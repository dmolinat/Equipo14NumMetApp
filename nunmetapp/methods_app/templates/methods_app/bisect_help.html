Are you ready to bisect functions? Click in Bisect.

THEORY. This method have some conditions:
  • \(f\) continuous in \([a, b]\).
  • (Intermediate Value Theorem): \(f(a)\)*\(f(b)\) \(\leq\) \(0\).
  • There is only one root in \([a,b]\).
PARAMS.
  • F_s: Function \(f\) to bisect.
  • A: Lower limit to bisect in [a,b].
  • B: Upper limit to bisect in [a,b].
  • Tolerance: Approximation accuracy figures or stopping bisect.

Example 1:

Supose that you want to find the root with accuracy of 6 figures of function \(f(x)=x^3+50x^2+4x-10\) in the interval \([0,0.5]\): Check if function satisfies IVT: \(f(0)f(0.5)\leq0\).

We write the next params in Bisect Form:

  • F_s: x**3+50*x**2+4*x-10
  • A: 0
  • B: 0.5
  • Tolerance: 1e-6
